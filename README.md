# CodeMentor AI

> AI coding mentor with **persistent memory** powered by Hindsight

Built for **HackWithIndia** — Theme: AI Agents That Learn Using Hindsight

---

## How Hindsight Memory Works

```
User submits code
      │
      ▼
  retain()  ──→  Hindsight stores fact (mistake / success)
                        │
              On next problem:
                        │
                        ▼
  recall()  ──→  Relevant past errors retrieved (top 5)
                        │
              Before AI responds:
                        │
                        ▼
  reflect() ──→  Personalized guidance generated by Hindsight
                        │
              Hindsight auto-consolidates observations over time
                        │
                        ▼
              [ Smarter, personalized mentoring ]
```

Every code submission, hint request, and chat interaction teaches the AI mentor
more about your specific weaknesses, patterns, and growth trajectory.

## Memory Flow

| Operation | When Called | What It Does |
|-----------|-------------|--------------|
| `retain()` | After every code submission + every hint | Stores user's mistake or success as a memory fact |
| `recall()` | On app load + on every problem select | Retrieves relevant past errors and patterns |
| `reflect()` | Before AI generates feedback | Creates personalized Hindsight-synthesized guidance |

## Design

The UI is modeled after VS Code with glassmorphism panels over a live Three.js
neural network scene. Color palette uses indigo/violet for the interface accent
and emerald green for memory/Hindsight indicators.

- **Font**: Inter (UI) + JetBrains Mono (code)
- **Primary accent**: `#6366f1` (indigo)
- **Memory accent**: `#10b981` (emerald)
- **Base**: `#0a0a0b` (near-black)
